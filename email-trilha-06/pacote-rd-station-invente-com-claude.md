# Pacote de disparo — Invente com Claude | E-mail 01

## Arquivo para importar

Use o arquivo `email-invente-com-claude-01.html` no editor HTML do RD Station. O botão já está configurado para a página do curso, com os parâmetros de campanha informados.

## Imagens da marca

Na pasta `assets`, estão os três arquivos que fazem parte do layout e não devem ser alterados: `logo_invente_com_claude.png`, `bonequinho-raio.png` e `passos-claude.png`.

Envie-os para a biblioteca de imagens do RD Station. Depois, no código HTML, substitua cada caminho que começa com `assets/` pela URL que o RD Station gerar para a respectiva imagem. As imagens são apoio visual; todo o conteúdo essencial continua como texto no HTML.

O e-mail não usa variável de nome no corpo. Se quiser personalizar, selecione o campo de nome na lista de variáveis do próprio RD Station apenas no campo **assunto**. A sintaxe pode variar conforme a conta e o editor; não cole literalmente o marcador abaixo — insira a variável pelo próprio RD Station.

## Assunto recomendado

`[Nome], o jeito mais simples de gastar menos tokens no Claude`

Ele promete um benefício prático e não começa vendendo. Caso prefira não usar personalização, use:

`O jeito mais simples de gastar menos tokens no Claude`

## Texto de prévia (preheader)

`Uma dica prática para pedir melhor, economizar tokens e obter respostas mais úteis.`

## Alternativas de assunto para teste A/B

1. `Antes de pedir mais ao Claude, faça isto`
2. `4 pontos para o Claude entender seu pedido de primeira`
3. `Menos tokens, menos retrabalho: uma dica para usar o Claude`

## CTA do botão

`Conhecer o Invente com Claude →`

## Checklist antes de enviar

- Troque o link provisório do botão pelo link da página do curso.
- Confirme se o cupom `INCC10` está ativo e se concede 10% de desconto.
- Configure o link de descadastro pelo próprio RD Station; ele não faz parte deste HTML.
- Envie um disparo de teste para Gmail e Outlook, no desktop e no celular.
- Não envie a arte original inteira como imagem: este HTML já leva o conteúdo principal como texto.
